== Description ==
Skype Addon is a wordpress plugin to add a skype call/chat button integration to page, post or even include in 
the theme file directly.

== Installation ==

To install JC Skype Addon:

1. Upload the 'jc-skype-addon' folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Administer in menu -> Skype Addon Settings
